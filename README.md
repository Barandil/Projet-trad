# Projet-trad
Projet de traduction pour magazine sportif

Le projet consistera à livrer une traduction française de la conference de démission de Marcelo Bielsa de son poste de sélélectionneur du Chilie. La commandandidaire est un magazine sportif. Cette traduction pourra être publiée ou bien être exploitée dans des articles. En outre, ce sera le directeur de rédaction du magazine qui aura le dernier mot concernant l'usage du livrable.

Ce projet se fera en plusieurs phases:

1-La première phase consistera à retranscrire la conférence prononcée en espagnol sur la base du contenu vidéo. 

2-La deuxième sera cela de la traduction dans un fichier réunissant la version originale et la traduction française.

3-La troisième sera celle de la relecture du correcteur bilingue.

Une fois cette relecture effectuée:

4-Etablir la version française dans un fichier distinct.

Une fois ces missions remplies, la version française sera présentée au directeur de rédaction pour validation et décision éditoriale.
